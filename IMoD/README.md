## Usage

* The `./example` directory contains three complete example datasets used in the paper: `energetics`, `BACE`, and `permeability`.

* To migrate to a custom dataset, place `Diffusion_v1.2.py`, `Sequence_v1.2.py`, and `D.csv` in the same directory.

* `TARGET_D` in `Diffusion_v1.2.py` and `Sequence_v1.2.py` represents the target property value expected for the generated molecules.

* `D.csv` must contain two columns: `SMILES`: molecular representation; `D`: target property.

> [!NOTE]
> The filename can be changed by modifying `DATA_CSV = "D.csv"`; the label column matching rule can be modified via `r"(^|[^A-Za-z])D([^A-Za-z]|$)|label|target"`.
>
> `chemprop` uses `v1.6.1`; the calling method differs between `Windows` and `Ubuntu`, and both `Diffusion_v1.2.py` and `Sequence_v1.2.py` support the two platforms.
>
> To avoid an oversized repository, the `./example` directory does not include `*.pt` files. If needed, the corresponding `*.pt` files can be obtained via email (misaraty@163.com) or regenerated by running the scripts.
