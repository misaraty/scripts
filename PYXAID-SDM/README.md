## PYXAID-SDM

Based on the `PYXAID` framework, this project develops `PYXAID-SDM`, an enhanced variant incorporating `SDM-based` decoherence correction. While preserving the original nonadiabatic molecular dynamics formalism and computational workflow, it improves the description of electronic decoherence processes, leading to better physical accuracy. `PYXAID-SDM` is suitable for excited-state carrier dynamics studies and serves as a foundation for further methodological developments.

## Usage

Replace `ElectronicStructure.cpp`, `ElectronicStructure.h`, and `namd.cpp` in the original `./src_cpp` directory, then recompile and install `PYXAID`.

Other usage remains the same as the original version.
